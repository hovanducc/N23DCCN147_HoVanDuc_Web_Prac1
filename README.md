# [Thực hành 1] - Xây dựng Blog Page với NextJS & Tailwind
Thông tin sinh viên
Họ và tên: Hồ Văn Đức
MSSV: N23DCCN147
Lớp: D23CQCN03-N
Giới thiệu
Blog Listing Page đơn giản xây dựng bằng Next.js (App Router) và Tailwind CSS, fetch dữ liệu từ JSONPlaceholder API, hỗ trợ xem chi tiết bài viết qua Dynamic Route và giao diện responsive trên mọi thiết bị.
Công nghệ:
Next.js  Tailwind CSS  TypeScript  JSONPlaceholder API
Chạy dự án:
  npm install
  npm run dev
